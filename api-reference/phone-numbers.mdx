## Overview

Phone numbers connect your agents to real telephone calls. Each number can be assigned:
- An **inbound agent** — handles calls that come in to this number
- An **outbound agent** — used as the caller ID when making outbound calls

<Info>
  TalkifAI uses BYOC (Bring Your Own Carrier) — you register numbers from your own telephony provider. See [Telephony Guide](/platform/telephony) for carrier setup.
</Info>

---

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/v1/phone-numbers` | List all phone numbers |
| `POST` | `/v1/phone-numbers` | Register a new phone number |
| `PUT` | `/v1/phone-numbers/{id}` | Update number details (SIP config) |
| `PATCH` | `/v1/phone-numbers/{id}` | Assign inbound/outbound agents |
| `DELETE` | `/v1/phone-numbers/{id}` | Remove a phone number |

## Register a Phone Number

```bash
POST /v1/phone-numbers
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

**Request body:**

```json
{
  "number": "+12025550100",
  "terminationUri": "myaccount.pstn.twilio.com",
  "sipUsername": "myaccount",
  "sipPassword": "secure_password_here"
}
```

| Field | Required | Description |
|-------|----------|-------------|
| `number` | Yes | Phone number in E.164 format (e.g., `+12025550100`) |
| `terminationUri` | Yes | Your carrier's SIP termination URI |
| `sipUsername` | No | SIP username for outbound authentication |
| `sipPassword` | No | SIP password for outbound authentication |

**Response:**

```json
{
  "success": true,
  "data": {
    "id": "pn_new456",
    "number": "+12025550100",
    "terminationUri": "myaccount.pstn.twilio.com",
    "inboundAgent": null,
    "outboundAgent": null,
    "createdAt": "2024-01-15T10:30:00Z"
  }
}
```

<Note>
  After registering, assign an inbound agent (see below) so the number can receive calls.
</Note>

## Update Number SIP Config

Update the SIP termination details for a number.

```bash
PUT /v1/phone-numbers/{id}
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

**Request body:**

```json
{
  "terminationUri": "newaccount.pstn.twilio.com",
  "sipUsername": "newusername",
  "sipPassword": "new_password"
}
```

## Assign Agents to a Number

Assign which agents handle inbound calls and which agent ID is used for outbound calls.

```bash
PATCH /v1/phone-numbers/{id}
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

**Request body:**

```json
{
  "inboundAgentId": "agent_xyz789",
  "outboundAgentId": "agent_xyz789"
}
```

| Field | Description |
|-------|-------------|
| `inboundAgentId` | Agent that handles incoming calls to this number. Set to `null` to unassign. |
| `outboundAgentId` | Agent used as the caller ID for outbound calls. Set to `null` to unassign. |

**Response:**

```json
{
  "success": true,
  "data": {
    "id": "pn_abc123",
    "number": "+12025550100",
    "inboundAgent": {
      "id": "agent_xyz789",
      "name": "Customer Support",
      "mode": "public"
    },
    "outboundAgent": {
      "id": "agent_xyz789",
      "name": "Customer Support",
      "mode": "public"
    }
  }
}
```

<Warning>
  Only **Pipeline** and **Realtime** agents can be assigned to phone numbers. Text agents do not support voice calls.
</Warning>

## Delete a Phone Number

```bash
DELETE /v1/phone-numbers/{id}
Authorization: Bearer YOUR_API_KEY
```

**Response:**

```json
{
  "success": true
}
```

<Warning>
  Deleting a phone number removes it from TalkifAI but does **not** release it from your carrier. You must release it separately in your carrier's dashboard.
</Warning>
